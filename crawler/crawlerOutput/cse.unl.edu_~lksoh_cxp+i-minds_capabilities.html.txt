<p class="MsoNormal"><o:p> </o:p></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>We built a suite
of I-MINDS Conference XP capabilities, from May 2005 to February 2006.<span style="mso-spacerun:yes">  </span>We grouped these capabilities as <a href="http://cse.unl.edu/~lksoh/cxp+i-minds/capabilities1.0.html">I-MINDS
Conference XP Capabilities 1.0</a>.<o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>In May 2006, we
released <a href="http://cse.unl.edu/~lksoh/cxp+i-minds/capabilities2.0.html">I-MINDS
Conference XP Capabilities 2.0</a>.<span style="mso-spacerun:yes">  </span>In
this version, we overhauled the way we handled <span class="SpellE">mySQL</span>
queries---improving the programmability of the software.<span style="mso-spacerun:yes">  </span>We also modified the interface significantly
to include more support for both teachers and students in reviewing the data
collected by I-MINDS.<o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>In December
2006, we released <a href="http://cse.unl.edu/~lksoh/cxp+i-minds/capabilities3.0.html">I-MINDS
Conference XP Capabilities 3.0</a>.<span style="mso-spacerun:yes">  </span>In
this version, we merged our main capabilities into one.<span style="mso-spacerun:yes">  </span>We also focused a lot of effort to address
some <a href="http://cse.unl.edu/~lksoh/cxp+i-minds/wireless.html">wireless
deployment</a> issues, including implementing a PGM protocol.<o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>In September
2007, we released <a href="http://cse.unl.edu/~lksoh/cxp+i-minds/capabilities4.0.html">I-MINDS
Conference XP Capabilities 4.0</a>.<span style="mso-spacerun:yes">  </span>In
this version, had several primary goals: 1) build a system robust enough to
operate in both wired and wireless environments,<span style="mso-spacerun:yes">  </span>2) extend the existing functionalities to
include 2 new modes – <span class="SpellE">StructuredChat</span> and <span class="SpellE">QuizMode</span>, 3) reorganize the agent functionalities into
specialties which are components that specialize in one area of tasks, 4)
create an XML configuration file to give users more flexibility in setting
options, 5) add network tracking for PGM and wireless environments, and 6) redo
the main CXP+I-MINDS form to offer more functionality and be more user
friendly.<span style="mso-spacerun:yes">  </span>All coding for 4.0 was moved
to Visual Studio 2005 to use .NET 2.0 and be compliant with CXP version
4.0.<span style="mso-spacerun:yes">  </span>Statistic tracking was also redone
to provide less reliance on database queries and faster performance, and the
statistical analysis tools were redone to offer better user interfaces and to
take advantage of the new tracking mechanisms.<span style="mso-spacerun:yes"> 
</span>Finally, <span class="SpellE">BuddyGroup</span> was renamed to <span class="SpellE">BuddyChat</span> to avoid confusion when talking about the
application and groups of buddies.<o:p></o:p></span></p>
