<p style="text-align:justify;font-size:12pt;color:white;">PROFESS is a biology database system that integrates databases describing PROtein Functions, Evolution, Structures and Sequences. PROFESS aims at assisting researchers in the functional and evolutionary analysis of the abundant number of novel proteins continually identified from whole-genome sequencing. <a class="faq_link" href="faq.php" title="Get more details">Read more...</a></p></div><br/>

<h4 class="index">Latest News</h4>
<div style="margin:20px;">
<p class="index_news">
<b>06/06/2010</b><br/>
			PROFESS has been published in the journal <i>Database</i>. Full-text article is available open-source <a class="faq_link" href="http://database.oxfordjournals.org/content/2010/baq011.full" target="_blank" title="Full-text">online</a>.
		</p><br/>
<p class="index_news">
<b>04/10/2010</b><br/>
			The <a class="faq_link" href="search.php" title="Advanced search">BLAST search</a> has been improved. You can now visualize the ouput from BLAST and select relevant protein sequences to further query PROFESS.
		</p><br/>
<p class="index_news">
<b>03/30/2010</b><br/>
			Release 1.5 now includes sequences from Swiss-PROT and TrEMBL from <a class="faq_link" href="http://www.uniprot.org/" target="_blank">UniProt</a>. This major update also includes <a class="faq_link" href="http://eggnog.embl.de/" target="_blank">eggNOG</a>, which replaces the older COG/KOG database and significantly increases the number of orthologous clusters.
		</p><br/>
<p class="index_news">
<b>02/21/2010</b><br/>
			Release 1.2 enables BLASTP queries against sequences within PROFESS. BLAST queries are available on the <a class="faq_link" href="search.php" title="Advanced search">Advanced Search</a> page.
		</p><br/>
<p class="index_news">
<b>08/12/2009</b><br/>
			The <a class="faq_link" href="http://www.essentialgene.org/" target="_blank">Database of Essential Genes</a> and the <a class="faq_link" href="http://dip.doe-mbi.ucla.edu/" target="_blank">Database of Interacting Proteins</a> are now part of PROFESS 1.1.
		</p><br/>
<p class="index_news">
<b>04/23/2009</b><br/>
			Release of PROFESS 1.0, which integrates <a class="faq_link" href="http://www.cathdb.info/" target="_blank">CATH</a>, <a class="faq_link" href="http://www.ncbi.nlm.nih.gov/COG/" target="_blank">COG</a>, <a class="faq_link" href="http://www.chem.qmul.ac.uk/iubmb/enzyme/" target="_blank">EC</a>, <a class="faq_link" href="http://www.essentialgene.org/" target="_blank">Essential Genes in <i>E. coli</i></a>, <a class="faq_link" href="http://www.geneontology.org/" target="_blank">GO</a>, <a class="faq_link" href="http://www.genome.jp/kegg/ligand.html" target="_blank">KEGG Ligands</a>, <a class="faq_link" href="http://www.rcsb.org/" target="_blank">PDB</a>, <a class="faq_link" href="http://pfam.sanger.ac.uk/" target="_blank">PFAM</a>, <a class="faq_link" href="http://genome.cshlp.org/content/16/5/686.abstract" target="_blank">Protein/Protein interactions in <i>E. coli</i></a> and <a class="faq_link" href="http://www.uniprot.org/taxonomy/" target="_blank">UniProtKB Taxonomy</a>. If you would like a specific database to be integrated within PROFESS, contact us!
		</p><br/>
<p class="index_news">
<b>04/07/2009</b><br/>
			First public release of PROFESS 0.7 (beta). Note this version is still under development and some features are missing. Please help us to improve the system by sending us your feedback and comments.
		</p><br/>
