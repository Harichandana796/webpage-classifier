<p class="MsoNormal"><b style="mso-bidi-font-weight:normal"><span style='font-family:"Eras Medium ITC";color:#FF6600'>5.1.<span style="mso-spacerun:yes">  </span>Introduction</span></b><span style='font-family:"Eras Medium ITC";color:#FF6600'><o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>To assist both
users and moderators analyze the data tracked by CXP+I-MINDS, we have designed
several tools:<span style="mso-spacerun:yes">  </span>1) VirtualClassroom, 2)
Statistics, 3) DatabaseSearch, and 4) QuizStats.<span style="mso-spacerun:yes">  </span>All four of these tools are available in the
Moderator version, while users only have Statistics and DatabaseSearch.
QuizStats is the only new analysis tool in version 4 of CXP+I-MINDS, but we
have modified VirtualClassroom to provide more statistical information based on
the increase in statistical tracking done in version 4.<span style="mso-spacerun:yes">  </span>In versions 2 and 3, VirtualClassroom had a
limited set of statistics, and also included an overview of all message traffic
from the system in the form of message and topic lists, but we removed these
because they were not used during deployment.<span style="mso-spacerun:yes"> 
</span>Also, for version 4, we included the Statistics tool in the user version
to replace its text-based predecessor, which was available in all prior
versions of CXP+I-MINDS, to accommodate the increase in statistical tracking
and to provide information in an easier to analyze format.<span style="mso-spacerun:yes">  </span>DatabaseSearch has remained unchanged from
its design in version 3.<o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>

modeling is done by each user agent based on their user’s interactions with
other users.<span style="mso-spacerun:yes">  </span>Moderators receive their
statistics from the user agents and the values stored in the database.<span style="mso-spacerun:yes">  </span>For version 4, we added new statistics, and
grouped the statistics by mode instead of by type as was done in previous
versions.<span style="mso-spacerun:yes">  </span>The current set of statistics

<p class="MsoNormal"><b style="mso-bidi-font-weight:normal"><span style='font-family:"Eras Medium ITC";color:#FF6600'>5.2.<span style="mso-spacerun:yes">  </span>Virtual Classroom Tool<o:p></o:p></span></b></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>The Virtual
Classroom form is very useful for moderators.<span style="mso-spacerun:yes"> 
</span>It displays in-depth statistical information about each user computed
over a given number of sessions (i.e., uses of the system).<span style="mso-spacerun:yes">  </span>Users are represented by small pie charts,
with each chart split into five pieces.<span style="mso-spacerun:yes"> 
</span>Each piece corresponds to their statistics for a specific mode (i.e.,
QuestionAnswering, BuddyChat, StructuredChat, Individual Quizzes, and
Collaborative Quizzes). <span style="mso-spacerun:yes"> </span>The pieces are
colored blue if the user is at or above the thresholds for those statistics (as
set by the moderator) and colored red if the user falls below the set
thresholds.<span style="mso-spacerun:yes">   </span>Furthermore, all red pie
slices are separated from the rest of the chart, and the more “bad” statistics
a user has, the further the slice is from the chart.<span style="mso-spacerun:yes">  </span>This system makes it easy for moderators to
quickly analyze and compare the performance of all users at a glance.<span style="mso-spacerun:yes">  </span>The thresholds for the statistics are stored
in the database so the moderators can set persistent levels and only modify
when desired.<span style="mso-spacerun:yes">  </span>An example of the Virtual
Classroom for can be seen in Figure 5.1.<o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>Clicking on a
user populates the list view on the right side with numerical values for each
statistic, with bad ones once again highlighted in red.<span style="mso-spacerun:yes">  </span>Also, clicking on a specific slice in a chart
loads the Statistics tool (described in the next section) with the proper
information for in-depth analysis.<span style="mso-spacerun:yes"> 
</span>Finally, hovering the mouse over any chart gives the user’s name and
identifier in a tooltip.<span style="mso-spacerun:yes">  </span>If a slice is
under the mouse, a listing of statistics that failed to meet the threshold is
added to the tooltip for easy identification.<o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>Whenever the
form is loaded or updated, the moderator agent checks the statistics for every
mode and tries to find modes that are especially problematic for most
users.<span style="mso-spacerun:yes">  </span>If it finds any such modes, it
alerts the moderator of such problems, as shown in Figure 5.1.<o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>

<p align="center" class="MsoNormal" style="text-align:center"><b style="mso-bidi-font-weight:
normal"><span style='font-family:"Eras Medium ITC"'>Figure 5.1:</span></b><span style='font-family:"Eras Medium ITC"'> <b style="mso-bidi-font-weight:normal">The
VirtualClassroom Tool with Problem Alert</b><o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><b style="mso-bidi-font-weight:normal"><span style='font-family:"Eras Medium ITC";color:#FF6600'>5.3.<span style="mso-spacerun:yes">  </span>Statistics Tool<o:p></o:p></span></b></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>The Statistic
form is used to display statistics information about users in bar charts for
easy comparison of values.<span style="mso-spacerun:yes">  </span>For each
mode, users and moderators can choose to display all the statistics of any
individual user or the values for each user for an individual statistic.<span style="mso-spacerun:yes">  </span>Once again, statistics that fall at or above
the thresholds set by moderators are shown in a blue color and those that fall
below thresholds are shown as a red color.<span style="mso-spacerun:yes"> 
</span>The thresholds are the same ones recorded in the database by the Virtual
Classroom form, and any change to the thresholds is propagated in both forms to
all users. Once again, the stats are computed for the number of sessions
specified in the bottom left corner of the screen, which is also tied to the
corresponding option in Virtual Classroom for moderators.<span style="mso-spacerun:yes">  </span>For an example of the Statistics Charts form,
please consult Figure 5.2.<o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>

<p align="center" class="MsoNormal" style="text-align:center"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><b style="mso-bidi-font-weight:normal"><span style='font-family:"Eras Medium ITC";color:#FF6600'>5.4.<span style="mso-spacerun:yes">  </span>Database Search</span></b><span style='font-family:"Eras Medium ITC";color:#FF6600'><o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>In order to
allow users and moderators to search through all of the message content
archived by CXP+I-MINDS, we have developed a tool called DatabaseSearch, shown
in Figure 5.3.<span style="mso-spacerun:yes">  </span>This tool allows for
searching for messages by any associated metadata, including when the message
was sent, what type of message it was, any keywords, etc.<span style="mso-spacerun:yes">  </span>For any found messages, a message tree can
also be constructed by right clicking on the message, then choosing to build a
message tree.<span style="mso-spacerun:yes">  </span>The tool also allows for
searching for different users in the system, which displays their name,
identifier, and the number of messages they have sent in the current
classroom.<span style="mso-spacerun:yes">  </span>Right clicking on a user
brings up an option to search for that user’s current buddies.<span style="mso-spacerun:yes">  </span>Moderators can also right click on a user to
bring up an option to view all messages sent by that user.<span style="mso-spacerun:yes">  </span>Users do not have this option because they
are only allowed to search for their own message.<span style="mso-spacerun:yes">  </span>This limitation is imposed because we don’t
want users to be able to find messages they otherwise wouldn’t have been able
to see through the QuestionAnswering, BuddyChat, and StructuredChat modes.<span style="mso-spacerun:yes">  </span><o:p></o:p></span></p>
<p align="center" class="MsoNormal" style="text-align:center"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>

<p align="center" class="MsoNormal" style="text-align:center"><b style="mso-bidi-font-weight:

<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><b style="mso-bidi-font-weight:normal"><span style='font-family:"Eras Medium ITC";color:#FF6600'>5.5.<span style="mso-spacerun:yes">  </span>QuizStats</span></b><span style='font-family:
"Eras Medium ITC";color:#FF6600'><o:p></o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'><o:p> </o:p></span></p>
<p class="MsoNormal"><span style='font-family:"Eras Medium ITC"'>The final
analysis tool provided by CXP+I-MINDS is QuizStats, shown in Figure 5.4, which
displays statistical information about quiz results.<span style="mso-spacerun:yes">  </span>Unlike the other analysis tools, which are
available from the main IMINDS form, QuizStats is loaded from the QuizMode
form.<span style="mso-spacerun:yes">  </span>This tool allows moderators to
choose from different assigned quizzes, organized by name and when they were
assigned.<span style="mso-spacerun:yes">  </span>It then displays important
statistical information about the quiz, such as the average, median, min, max,
and standard deviation of score, average duration, and which questions were
answered correctly and incorrectly the most often.<span style="mso-spacerun:yes">  </span>The form also displays the score given to each
user if it was an individual quiz, or to each group if it was a collaborative
quiz.<span style="mso-spacerun:yes">  </span>The form also provides a bar chart
showing how many users or group answered each question correctly.<span style="mso-spacerun:yes">  </span>Finally, the form provides a listing of all
questions in the quiz. <span style="mso-spacerun:yes"> </span>If a moderator
selects a specific question, the form changes to show which answers each user
or group selected (with wrong answers highlighted in red), and the bar chart
shows how many people selected each possible answer, with correct answers
colored blue and incorrect answers colored red.<o:p></o:p></span></p>
